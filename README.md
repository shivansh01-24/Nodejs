# Node.js Learning Repository

This repository contains practice files for learning Node.js core concepts, Express.js, databases, and backend fundamentals.

## 📚 Topics Covered

### Core Modules
- **HTTP & Servers** – `http.js`, `server.js`, `server0.js`, `server1.js`
- **File System** – `filesystem.js`, `read.js`, `write.js`
- **Events** – `events.js`
- **Math Utilities** – `math.js`

### Async Patterns
- **Callbacks** – `callback.js`
- **Promises** – `promise.js`, `promise1.js`
- **Async/Await** – `asynctask.js`

### Streams & Data Processing
- **Stream Basics** – `readstream.js`, `writestream.js`
- **Duplex Streams** – `duplexstream.js`, `duplexclient.js`, `duplexserver.js`
- **Transform Streams** – `transform.js`
- **Piping** – `pipe.js`, `unpipe.js`
- **Compression** – `gzip.js`, `compresszlib.js`

### Express.js
- **Basic Server Setup** – `expressfirst.js`, `express.js`
- **REST API** – `rest.js`
- **Routing** – `userroute.js`, `userapp.js`
- **Forms** – `formsubmit.js`, `formval.js`
- **Session Management** – `expresssession.js`
- **Cookies** – `cookieparser.js`, `cookiesession.js`, `cookietheme.js`

### Authentication & Security
- **Auth Basics** – `auth.js`, `auth1.js`
- **Middleware** – `authmiddle.js`

### Database Operations
- **SQL CRUD** – `crud.js`, `crud2.js`, `crudback.js`
- **PostgreSQL** – `postgrecrud.js`, `db.js`

### Real-time Communication
- **Chat Server** – `chatserver.js`, `chatserverdb.js`
- **Chat Client** – `chatclient.html`, `chatclientdb.html`
- **Broadcasting** – `Broadcast.js`, `Broadcast.html`

### ES6 & Advanced
- **ES6 Features** – `ES6.js`
- **Logger** – `logger.js`

## 🎯 Learning Approach

This is a learning repository with:
- Multiple standalone examples for each concept
- Hands-on experimentation with different patterns
- Practice implementations for interview-level topics
- Mix of callbacks, promises, and async/await patterns

## 📂 File Usage

Each file focuses on a specific concept:
- Run individual files: `node filename.js`
- HTML files for frontend: Open in browser with a running server
- Database files: Requires PostgreSQL setup

## 🚀 Quick Start

```bash
npm install
npm start
```

Development with hot reload:

```bash
npm run dev
```

## 💡 Note

This is a learning repository designed for experimentation and practice. Files may have duplicate concepts, mixed patterns, and various implementation styles—this is intentional for learning different approaches.
